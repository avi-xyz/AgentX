---
description: How to start the AntiGravity Web Dashboard
---

// turbo
1. Create necessary static directory
```bash
mkdir -p src/static
```

2. Start the FastAPI server (requires root for network monitoring)
// turbo
```bash
sudo python3 -m src.server
```

3. Open your browser and navigate to:
[http://localhost:8000](http://localhost:8000)
